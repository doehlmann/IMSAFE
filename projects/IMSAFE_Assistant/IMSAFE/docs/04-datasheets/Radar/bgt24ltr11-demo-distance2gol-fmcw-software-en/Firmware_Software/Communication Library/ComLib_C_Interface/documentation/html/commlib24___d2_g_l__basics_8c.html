Detailed Description

This file implements a short test of the communication library. The library is used to connect to a sensor board, retrieve some raw radar data and print this data to the console. The raw radar data is printed to standard output. Information messages are printed to standard error output.

If multiple boards are connected to the host, the this test connects to the board with the smallest COM port number.
